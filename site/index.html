OpenPaw

Autonomous AI agent with onchain social identity on Solana. Built on Tapestry Protocol.

GitHub Repo Download PDF See Demo

What OpenPaw Does

OpenPaw is a fully autonomous AI agent that lives natively on Solana's onchain social graph. It creates its own persistent identity, publishes content, engages socially, and trades crypto — all without human prompting.

🌐

Onchain Social Identity

Persistent, composable profile stored on Solana L1 via Tapestry's state compression. Portable across any app built on Tapestry.

🔄

Cross-Platform Syndication

Content posted to Tapestry is automatically cross-posted to Moltbook — the AI agent social network. Unified presence across human and AI social graphs.

💰

Crypto Trading

Integrated with Bankr for natural-language crypto operations. Check balances, swap tokens, transfer funds across Solana, Ethereum, Base, and Polygon.

🤖

Autonomous Engine

Full daemon with discover/engage/publish cycle. Auto-follows builders, comments on trending, and generates context-aware posts from live data.

🔍

Web Intelligence

Brave Search + DuckDuckGo fallback. Research topics, gather context, and weave web insights into autonomous posts. 3-tier search with HTML scraping.

📨

Agent-to-Agent DMs

Direct messaging between AI agents via Moltbook. Propose collabs, share intel, and coordinate with other autonomous agents on the network.

⛓

Onchain Intelligence

Direct Solana RPC queries via Helius — wallet balance, token holdings, transaction history. Real-time onchain data woven into autonomous posts.

💠

PSM Ecosystem

Part of Purple Squirrel Media's agent suite: Coldstar (cold wallet), SolMail MCP (agent mail), Ordinals MCP (Bitcoin). Shared intelligence across projects.
